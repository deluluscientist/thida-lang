Thida Lang
Myanmar Grammar Based Programming Language

Thida Lang is the first programming language that follows Myanmar's natural Subject–Object–Verb sentence structure. Write data analysis programs that read like plain Burmese text.

Get Started → Our Story View on GitHub

Why Thida Lang?

Built for Myanmar speakers who want to work with data naturally.

🇲🇲
Myanmar Grammar Syntax
Every statement ends with a Burmese verb. Code reads like natural Burmese sentences — no English grammar required.
📊
Built for Data Analysis
Load CSV & Excel, compute statistics, clean data, train linear models, and generate charts — all in Myanmar syntax.
🖥
4-Panel IDE
RStudio-inspired web IDE with code editor, data viewer, console, and plots panel. Dark and light themes.
📈
5 Chart Types
Scatter plot, histogram, bar chart, line chart, and box plot — with automatic statistical descriptions.
🗃
Built-in Datasets
Four ready-to-use datasets: students, sales, health, and housing — all with Myanmar context.
📖
9 Lesson Docs
Complete learning path from getting started to linear regression, all written in Myanmar language.

Quick Start

Get up and running in under a minute.

1. Install dependencies

bash
pip install -r requirements.txt

2. Start the IDE

bash
python3 web/app.py # → http://localhost:5050

3. Write your first program

thida
// Hello World ("Hello, Thida Lang!") ကိုဖော်ပြပါ // Load dataset ("students") ကိုဖတ်ပါ ("students") ကို Header ဖော်ပြပါ // Draw a scatter plot ("students" > x = "Math" , y = "Science") ကို scatter_plot အဖြစ် ပုံဆွဲပေးပါ

Language at a Glance

Every statement: (object) verb

Output
("မင်္ဂလာပါ!") ကိုဖော်ပြပါ (42) ကိုဖော်ပြပါ
Variables
(95) ကို score ဟု သတ်မှတ်ပါ (score) ကိုဖော်ပြပါ
Statistics
("students" > column = "Math") ကို mean ကိုဖော်ပြပါ
Filter
("students" > column = "Math" , value = 70 , op = ">=") ကို စစ်ထုတ်ပါ
Visualization
("sales" > x = "Region" , y = "Revenue") ကို bar_chart အဖြစ် ပုံဆွဲပေးပါ
ML Model
("housing" > features = "Size_sqft,Bedrooms" , target = "Price_MMK") ကို linear_model တည်ဆောက်ပါ
