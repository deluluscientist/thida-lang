Common Questions

Can I contribute to Thida Lang?
Yes! The project is open source under MIT license. Fork the repo on GitHub, make your changes, and open a pull request. All contributions are welcome — code, documentation, translations, or testing.
I found a bug. How do I report it?
Open an issue on the GitHub Issues page with a description of the problem, the code you ran, and the error message you received.
Is Thida Lang free to use?
Completely free and open source under the MIT license. You can use it, modify it, and distribute it freely.
Will there be a version that runs in the browser without installing anything?
It's on the roadmap. For now, Thida Lang requires a local Python install to run the interpreter. A hosted version is planned for a future release.
Can I use Thida Lang for my own research or teaching?
Absolutely. If you do, we'd love to hear about it — send a message to the email above.
