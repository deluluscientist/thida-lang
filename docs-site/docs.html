Installation

Thida Lang requires Python 3.9 or newer.

1. Clone the repository

bash
git clone https://github.com/deluluscientist/thida-lang.git cd thida-lang

2. Install dependencies

bash
pip install -r requirements.txt

This installs: pandas numpy openpyxl scikit-learn matplotlib flask

Quick Start

thida
// Print a message ("Hello, Thida Lang!") ကိုဖော်ပြပါ // Load the students dataset ("students") ကိုဖတ်ပါ ("students") ကို Header ဖော်ပြပါ ("students") ကို အကျဥ်းချုပ် ဖော်ပြပါ // Compute the mean of Math scores ("students" > column = "Math") ကို mean ကိုဖော်ပြပါ // Draw a scatter plot ("students" > x = "Math" , y = "Science") ကို scatter_plot အဖြစ် ပုံဆွဲပေးပါ

Web IDE

bash
python3 web/app.py # Open http://localhost:5050

The IDE has four panels:

Data ViewerTop-rightInspect loaded datasets as tables
ConsoleBottom-leftOutput, errors, run status
PlotsBottom-rightCharts with statistical descriptions

Keyboard shortcuts: Ctrl+Enter run · Ctrl+S save · Ctrl+B toggle sidebar

Syntax Design

Thida Lang follows Myanmar's Subject–Object–Verb (SOV) grammar. Every statement places the object inside parentheses and ends with a Myanmar verb.

pattern
(object) verb (object > key = value , key = value) verb
💡 The > pipe separates the dataset name from its options. Commas separate multiple options.

Output

Print any value — string, number, or variable.

thida
("မင်္ဂလာပါ!") ကိုဖော်ပြပါ (42) ကိုဖော်ပြပါ (my_variable) ကိုဖော်ပြပါ

Variables

thida
// Assign a value (95) ကို score ဟု သတ်မှတ်ပါ ("Yangon") ကို city ဟု သတ်မှတ်ပါ // Display a variable (score) ကိုဖော်ပြပါ // Save dataset under a new name (students) ကို backup အဖြစ် သိမ်းဆည်းပါ

Loading Data

thida
// Built-in dataset (by name) ("students") ကိုဖတ်ပါ // CSV file ("data.csv" , file type = csv , safe copy = true) ကိုဖတ်ပါ // Excel file ("report.xlsx" , file type = excel , Sheet = "Sheet1") ကိုဖတ်ပါ
ℹ️ The dataset is stored automatically under the filename stem — "students.csv" becomes the variable students.

Data Inspection

thida
// Column names ("students") ကို Header ဖော်ပြပါ // Summary statistics ("students") ကို အကျဥ်းချုပ် ဖော်ပြပါ // First N rows ("students") ရဲ့ အစပိုင်း "10" လိုင်းဖော်ပြပါ // Column data types ("students") ရဲ့ အမျိုးအစားများ ဖော်ပြပါ // Count missing values ("students") ရဲ့ missing values အရေအတွက်ကို ဖော်ပြပါ

Statistics

OperationSyntax
Mean("dataset" > column = "col") ကို mean ကိုဖော်ပြပါ
Maximum("dataset" > column = "col") ကို max ကိုဖော်ပြပါ
Minimum("dataset" > column = "col") ကို min ကိုဖော်ပြပါ
Mode("dataset" > column = "col") ကို mode ကိုဖော်ပြပါ
Frequency("dataset" > column = "col") ကို frequency ကိုဖော်ပြပါ
Group By("dataset" > group by "col" , agg = "sum" , target = "col2") ကို ဖော်ပြပါ
example
("students" > column = "Math") ကို mean ကိုဖော်ပြပါ ("students" > column = "Math") ကို max ကိုဖော်ပြပါ ("sales" > column = "Region") ကို frequency ကိုဖော်ပြပါ ("sales" > group by "Region" , agg = "sum" , target = "Revenue") ကို ဖော်ပြပါ

Data Cleaning

thida
// Fill missing values (mean / median / mode / value) ("students" > missing values = "mean" , column = "Math") ကို ဖြည့်ပေးပါ // Drop rows with any missing value ("students") ကို ဖျက်ပစ်ပါ // Drop a column ("students" > column = "StudentID") ကို ဖျက်ပစ်ပါ // Rename a column ("sales" > old = "Rev" , new = "Revenue_USD") ကို ပြောင်းပေးပါ // Filter rows ("students" > column = "Math" , value = 70 , op = ">=") ကို စစ်ထုတ်ပါ // Sort descending ("students" > column = "Math" , order = "descending") ကို စီပေးပါ // Add a new column ("sales" > name = "Tax" , expr = "Revenue * 0.1") ကို ထည့်ပေးပါ // Save to CSV ("students" > filename = "output.csv") ကို သိမ်းဆည်းပါ
